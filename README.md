# F1 Telemetry Pro Analysis 🏎️
Ph.D grade research tool for Formula 1 data analysis using FastAPI, React, and Groq LLM.

## Features
- **Real-time API Sync**: Integrates with OpenF1 API.
- **Comparative Analysis**: Compare speeds between any two drivers.
- **AI Engineering Verdict**: Uses Llama 3.3 to provide technical insights into driver performance.

## Tech Stack
- **Backend**: Python (FastAPI), SQLAlchemy, Groq API.
- **Frontend**: React (Vite), Tailwind CSS, Recharts, Lucide Icons.
